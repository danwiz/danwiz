# Code of Conduct

This project is committed to a respectful, inclusive, and professional collaboration environment.

Expected behavior includes constructive feedback, technical rigor, patience with learners, respect for differing viewpoints, and responsible handling of security or privacy concerns.

Harassment, discriminatory conduct, personal attacks, deliberate disruption, and disclosure of private information are not acceptable.

Project maintainers may clarify standards, remove inappropriate content, restrict participation, or take other proportionate action to protect the community.
